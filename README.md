# Suppl'Eye Website

Website for Suppl'Eye - Revolutionizing Product Imagery.

## Project Structure

- `index.html`: Main HTML file.
- `style.css`: Main stylesheet.
- `images/`: Directory for image assets.
- `logos/`: Directory for logo assets.

## Technologies

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties)
- Font: Manrope

## Setup

Just open `index.html` in a web browser to view the site.
